## ミズナ観察日記：発芽から栽培終了までの記録

### 1. 種植え：水耕栽培の準備と発芽のコツ

水耕栽培のスタートは種植えからです。こちらのブログを参考に、100均のケースとスポンジ、保温遮熱シートで自作プランターを作成しました。スポンジに水をたっぷり含ませ、乾燥を防ぐためにトイレットペーパーを被せるのがポイントです。

{{< blog-card "https://agri.mynavi.jp/2018_01_23_16792/" >}}

*   **ミズナの種植えと発芽記録**

{{< self-blog-card "article/posts/2019-05-06-diary1" >}}

### 2. 11日目：初期成長と本葉の展開

種植えから11日目。ミズナは5cmほどに成長し、小さな本葉も確認できるようになりました。この時期の成長は、毎日の楽しみです。

{{< self-blog-card "article/posts/2019-05-17-diary2" >}}

### 3. 16日目：水耕栽培システムへの移植

少し早いと感じましたが、より本格的な**水耕栽培システム**へ苗を移植しました。この移植が、その後の成長にどう影響するのか期待が高まります。

{{< self-blog-card "article/posts/2019-05-22-diary3" >}}

### 4. 29日目：ミズナらしい姿へ変化

移植から約2週間、栽培開始から29日目には、ミズナの特徴的なギザギザとした葉の形がはっきりとわかるようになりました。

{{< self-blog-card "article/posts/2019-06-04-diary4" >}}

### 5. 38日目：日光を求めるミズナたち

栽培38日目。ミズナたちが一斉に日光の方向を向いている姿は、とても可愛らしく見えます。太陽のエネルギーを最大限に吸収しようとしているのが伝わってきます。

{{< self-blog-card "article/posts/2019-06-13-diary5" >}}

### 6. 41日目：収穫間近！

栽培41日目には、ミズナは十分に成長し、もういつ収穫しても良さそうな状態になりました。瑞々しい葉が、収穫の喜びを予感させます。

{{< self-blog-card "article/posts/2019-06-16-diary6" >}}

### 7. 45日目：収穫計画

栽培45日目。いよいよ本格的な収穫の準備です。次の休日に一斉収穫を行うことに決めました。

{{< self-blog-card "article/posts/2019-06-20-diary7" >}}

### 8. 47日目：感動の初収穫！

ついに**水耕栽培システムの初収穫**です！この日を待ちわびていました。収穫したばかりの新鮮なミズナは、シンプルにサラダで美味しくいただきました。自分で育てた野菜の味は格別です。

{{< self-blog-card "article/posts/2019-06-22-diary8" >}}

### 9. 53日目：再生栽培への挑戦

ミズナは根を残しておくと**再生する**と聞き、引き続き栽培を続行しました。小さな新芽が再び育ち始めるのを楽しみに、観察を続けます。

{{< self-blog-card "article/posts/2019-06-28-diary9" >}}

### 10. 55日目：2回目の収穫！

再生したミズナも再び大きく育ち、2回目の収穫を行いました！今回はスープに入れて、ミズナの風味を存分に味わいました。

{{< self-blog-card "article/posts/2019-06-30-diary10" >}}

### 11. 58日目：さらなる再生は？

残っていたミズナもほとんど収穫しました。ここからさらに育つかは実験的な試みです。

{{< self-blog-card "article/posts/2019-07-03-diary11" >}}

### 12. 65日目：成長の限界と次のステップ

種から育て始めて約2ヶ月。ミズナは枯れてはいませんが、成長が止まっているように見えます。植物にも寿命があることを実感します。

{{< self-blog-card "article/posts/2019-07-10-diary12" >}}

### 13. 68日目：栽培終了と感謝

根本が枯れてきたため、ついにミズナを撤去しました。最後まで収穫できたミズナに感謝です。

{{< self-blog-card "article/posts/2019-07-13-diary13" >}}

## 【まとめ】水耕栽培でミズナを育ててみた感想と今後の改善点

初めての水耕栽培でしたが、ミズナの育成を通じて多くの感動と学びがありました。

> **水耕栽培でミズナを育ててみた感想**
>
> *   何気なく始めた水耕栽培ですが、収穫できると達成感があった。
> *   毎日成長していくのが楽しみになっていた。
> *   自分で栽培した物を食べることへの充実感があった。
> *   まだまだ、改善できる点があるなと思った。

水耕栽培は、手軽に始められ、日々の成長を楽しめる素晴らしい家庭菜園の方法だと感じています。今後は、さらに効率的な栽培方法や装置の改善に挑戦し、より豊かな収穫を目指していきたいと思います。
